# Mars Rover Problem

## Run Script

1. npm install
2. npm start \<upper-right coordinates\> \<start position\> \<instructions\>
   - Test Input 1: `npm start "5 5" "1 2 N" "LMLMLMLMM"`
   - Test Input 2: `npm start "5 5" "3 3 E" "MMRMMRMRRM"`

## Technologies Used

- NPM
- Node.js
- TypeScript
- Jest (Unit Test)
